# GitHub Pages Upload

Upload these files to the root of your GitHub repo:

- `index.html`
- `styles.css`
- `app.js`
- `manifest.json`

Then update Apps Script with the current `Code.gs`, save, and deploy a new web app version.

After that:
1. Open your GitHub repo on GitHub.
2. Upload the four files from this folder.
3. Go to `Settings` -> `Pages`.
4. Set source to `Deploy from a branch`.
5. Use branch `main` and folder `/root`.
6. Wait for the GitHub Pages URL.
7. Open that URL in Safari.
8. Tap `Share` -> `Add to Home Screen`.
